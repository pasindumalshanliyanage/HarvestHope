Technical Description

The goal was to design a consistent user experience with modern HTML5 structure and responsive styling using CSS Flexbox. All pages created by Student 01 use:

  • HTML5 Semantic Tags: Proper usage of header, section, nav, article, and footer.
  • Card-Based Layouts: Information presented using visually distinct boxes with shadows and hover effects.
  • Responsive Design: Media queries and flexible units used to adapt the layout to various devices.
  • Interactive Features: Review forms, hover transitions, and button styling for enhanced engagement.

Challenges & Lessons Learned

  • Consistency: Maintaining design uniformity across all pages using the global CSS file was critical.
  • User Engagement: Designing interactive volunteer and feedback sections required testing for ease of use.
  • Responsiveness: Ensuring visual balance on both large and small screens was refined using Flexbox.
  • Content Relevance: Focused on meaningful topics such as Backyard Farming to strengthen the Zero Hunger theme.
