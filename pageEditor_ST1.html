Splash Screen

The splash screen serves as the welcoming entrance to the HarvestHope website, designed to make a strong first impression on users. It features the website name and introduces team members through clean, visually engaging design and smooth transitions.

Key HTML Elements

  • Used a full viewport container with a background image and overlay to highlight content.
  • Included semantic tags such as <header> and <section> for structure and accessibility.

Key CSS Techniques

  • Applied flexbox for vertical and horizontal centering of content.
  • Used overlay gradients to improve text readability over background images.
  • Implemented fade-in animations to enhance user engagement.

Design Decisions

The color scheme emphasizes warmth and hope, using the accent orange (#D8931D) to highlight text and interactive elements. The splash screen balances visual impact with simplicity to avoid overwhelming first-time visitors.

Technical Description

The primary goal was to build a consistent, user-friendly web experience using modern HTML5 semantics combined with CSS Flexbox for responsive layouts. Key technical aspects include:

  • HTML5 Semantic Elements: Proper use of elements like <header>, <section>, <nav>, <article>, and <footer> enhances accessibility and SEO.
  • Card-Based Design: Information is displayed in distinct card components with shadow effects and hover interactions to improve readability and visual engagement.
  • Responsive Styling: Media queries allow flexible adjustment of layout and typography across devices from desktop to mobile.
  • Interactive Features: Review forms with validation, hover transitions, and styled buttons provide dynamic user interaction while maintaining usability.

All pages adhere to best practices for maintainability and scalability, facilitating team collaboration and future updates.

Challenges & Lessons Learned

  • Consistency: Maintaining design uniformity across multiple pages was crucial. Using a global stylesheet helped ensure cohesive styling and easier maintenance.
  • User Engagement: Creating interactive volunteer and feedback forms required extensive testing to ensure ease of use and accessibility.
  • Responsiveness: Adapting complex layouts to work seamlessly on different screen sizes was challenging but essential to provide an optimal experience on mobiles and tablets.
  • Content Relevance: Emphasizing meaningful topics such as sustainable backyard farming strengthened the website’s message aligned with the Zero Hunger goal.
  • Time Management: Coordinating team roles and adhering to deadlines reinforced project management skills.

Accessibility

Accessibility considerations are integrated throughout the site, including:

  • Use of semantic HTML5 elements to improve screen reader navigation.
  • High color contrast ratios between text and backgrounds to aid readability.
  • Keyboard navigability ensured for all interactive elements such as links and forms.
  • Alt attributes for all images providing descriptive text for visually impaired users.
  • Responsive typography that scales appropriately to user device settings.

These measures ensure compliance with WCAG 2.1 Level AA standards and promote inclusivity.

Compliance

The website is designed to meet the following standards and guidelines:

  • W3C HTML5 and CSS3 Validation: All code passes validation checks to ensure structural correctness and browser compatibility.
  • WCAG 2.1 Level AA Accessibility: Incorporates features to support users with disabilities and assistive technologies.
  • Cross-Browser Compatibility: Tested across major browsers (Chrome, Firefox, Edge, Safari) for consistent behavior.
  • Responsive Web Design Principles: Ensures usability on a range of devices and screen sizes.

Adherence to these standards guarantees a robust, user-friendly, and professional web presence.
